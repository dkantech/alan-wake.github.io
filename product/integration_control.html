제품 종류
01
통합 제어기(LAN 8Port)
DKRTU-L8
- 다수의 통신 프로토콜 제공, I/O 보드 코드 출력
- 주변 장치의 통합 기능 수행
01
Audit Unit
  • - 4개 출력(채널별 30w), 각 채널 Line Out 변경
  • - 비상벨 마이크 장치와 연동한 양방향 인터폰 통화
  • - 외부 오디오 장치 출력(Line Input)
  • - Nand Flash 또는 SD Card Slot의 음원 파일 저장
  • - MCU와 연동한 저장 음원의 선택적인 방송 송출
02
Serial Unit
  • - RS-232 Port
  • - RS-422 및 485 Port
  • - 각 Port 별 통신 속도 및 흐름 설정
03
Ethernet Unit
  • - 2 Port SFP
  • - 8 Port 10/100/1000Mbps 자동 접속
  • - Port 별 자동 전원 공급(PoE, PoE+)
  • - 상태 LED 제공
  • - FAN 구동 회로 내장으로 과열 시 기기 냉각
04
I/O Unit
  • - 8개 Input Port를 이용한 각종 센서 연동(N.C / N.O)
  • - 8개 OutPut Port의 Relay 접점을 이용한 연동장비 제어
  • - I/O 운영 프로그램에 의한 설정 및 제어
분류 규격 비고
전원 입력 전원 90~264AVC, 50/60Hz, MAX 5A
PoE 전원 100W 2개
System& 전원 100W 2개
M.C.U Clock 72MHz
Ethernet Module 10/100Mbps
Serial RS-232 전송 방식 Full Duplex 2 Port
도달 거리 15M
최고 전송속도 20Kb/s
최대 입력 전압 +15V
RS-422 전송 방식 Full Duplex 2 Port 선택 사용
도달 거리 1.2Km
최고 전송속도 10Kb/s
최대 입력 전압 -7V +7V
RS-485 전송 방식 Half Duplex
도달 거리 1.2Km
최고 전송속도 10Kb/s
최대 입력 전압 -7V +12V
Max Drivers 32
I/O Port Input(8CH) Forward Current : 20mA 이상 스위치 접점
OutPut(8CH) N.O / N.C(Max: AC250V 5A) 릴레이 접점
Audio AMP Audio AMP(4CH) 30W x 4CH = 120W, 8Ω (Class-D), 사용전압 : 24V
Line Input (1CH) 600Ω -10dB 이상 (Balanced), 입력 주파수 : 20Hz ~ 20KHz
MIC Input (PTT 1CH) -55dB 이상, 입력 주파수 : 100Hz ~ 10KHz
IP Network 10/100/1000/Base-T 8 Port, RJ-45, Auto-negotiation, PoE / PoE+ 1 Port 내부용
Gigabit Ethernet 2 Port, 1G / 1.25Gbps SFP
