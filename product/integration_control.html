제품 종류
01
통합 제어기(LAN 8Port)
DKRTU-L8
- 다수의 통신 프로토콜 제공, I/O 보드 코드 출력
- 주변 장치의 통합 기능 수행
01
Audit Unit
  • - 4개 출력(채널별 30w), 각 채널 Line Out 변경
  • - 비상벨 마이크 장치와 연동한 양방향 인터폰 통화
  • - 외부 오디오 장치 출력(Line Input)
  • - Nand Flash 또는 SD Card Slot의 음원 파일 저장
  • - MCU와 연동한 저장 음원의 선택적인 방송 송출
02
Serial Unit
  • - RS-232 Port
  • - RS-422 및 485 Port
  • - 각 Port 별 통신 속도 및 흐름 설정
03
Ethernet Unit
  • - 2 Port SFP
  • - 8 Port 10/100/1000Mbps 자동 접속
  • - Port 별 자동 전원 공급(PoE, PoE+)
  • - 상태 LED 제공
  • - FAN 구동 회로 내장으로 과열 시 기기 냉각
04
I/O Unit
  • - 8개 Input Port를 이용한 각종 센서 연동(N.C / N.O)
  • - 8개 OutPut Port의 Relay 접점을 이용한 연동장비 제어
  • - I/O 운영 프로그램에 의한 설정 및 제어
02
통합 제어기(LAN 4Port)
DKRTU-L4
- 전파(全波) 측정 방식의 누설전류 검출 메커니즘으로 오류 최소화
- 써지보호기 일체형으로 SPD 40kA 이상 결합 가능(별도의 SPD를 설치할 필요 없음)
- 일반 차단기와 같이 다양한 장소에 적용 가능
01
Audit Unit
  • - 2개 출력(채널 별 30W), 각 채널 Line Out 변경
  • - 비상전화 마이크 장치와 연동한 양방향 인터폰 통화
  • - 외부 오디오 장치 출력 (Line Input)
  • - Nand Flash 또는 SD Card Slot의 음원 파일 저장
  • - MCU와 연동한 저장 음원의 선택적인 방송 송출
02
Serial Unit
  • - RS-232 : 1Port
  • - RS-422 및 485 Port : 1Port
  • - 각 Port 별 통신 속도 및 흐름 설정
03
Ethernet Unit
  • - 2 Port, SFP
  • - 4 Port, 10/100/1000Mbps 자동 접속
  • - Port 별 자동 전원 공급(PoE, PoE+)
  • - 상태 LED 제공
04
I/O Unit
  • - 4개 Input Port를 이용한 각 종 센서 연동(N.C / N.O)
  • - 4개 Output Port의 Relay 접점을 이용한 연동장비 제어
